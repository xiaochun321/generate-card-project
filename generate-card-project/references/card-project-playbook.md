# Card Project Playbook

## Source Pattern

The original idiom-card project produced 152 vertical cards at `1530x2741`. The winning architecture was:

- text layer: title, pinyin, idiom, explanation, story, tip rendered exactly with Pillow
- illustration layer: generated by AI, cached locally, checked and retried
- base layer: one fixed parchment-style background shared by all cards

Dependencies:

```bash
pip install Pillow numpy httpx openai python-dotenv pypinyin
```

Expected API environment:

```env
AGNES_API_KEY=your-api-key
AGNES_BASE_URL=https://apihub.agnes-ai.com/v1
```

## Layout Constants

Use percentages from the canvas size instead of hard-coded absolute positions:

```python
W, H = 1530, 2741
top_title_y = int(H * 0.045)
pinyin_y = int(H * 0.105)
idiom_y = int(H * 0.125)
```

Idiom cell sizing:

```python
if n_chars <= 4:
    cell_width = int(W * 0.16)
    idiom_size = int(W * 0.12)
    pinyin_size = int(W * 0.034)
elif n_chars == 5:
    cell_width = int(W * 0.135)
    idiom_size = int(W * 0.10)
    pinyin_size = int(W * 0.030)
else:
    cell_width = int(W * 0.12)
    idiom_size = int(W * 0.088)
    pinyin_size = int(W * 0.026)
```

Center each pinyin syllable over its own cell:

```python
pinyin_parts = pinyin.split()
for i, py in enumerate(pinyin_parts[:n_chars]):
    cell_center_x = start_x + i * cell_width + cell_width // 2
    pw, _ = measure_text(font_pinyin, py)
    draw.text((cell_center_x - pw // 2, pinyin_y), py, font=font_pinyin, fill=COLOR_PINYIN)
```

Move Chinese characters upward by `int(text_height * 0.10)` inside the cell to correct visual baseline bias.

## Adaptive Text Fitting

Start from generous sizes and shrink until the estimated section height fits:

```python
best_label_px = int(W * 0.038)
best_body_px = int(W * 0.032)
min_label_px = int(W * 0.018)
min_body_px = int(W * 0.015)

while best_label_px > min_label_px and best_body_px > min_body_px:
    needed = estimate_total_height(best_label_px, best_body_px)
    if needed <= available_h:
        break
    best_label_px -= 2
    best_body_px -= 2
```

The estimate must include all section labels, wrapped body lines, line height, and paragraph gaps.

## Illustration Prompt

Avoid visible text by not putting the idiom/title into the prompt. Prefer a mood/story prompt:

```python
prompt = (
    "Watercolor and ink illustration, ancient historical scene. "
    f"Mood and story: {explanation[:60]}. "
    "Traditional costume, natural architecture, classical composition, "
    "warm sepia beige parchment tones, aged paper texture background, "
    "soft watercolor brushwork, delicate details, horizontal wide composition. "
    "CRITICAL - DO NOT DRAW: absolutely no text, no letters, no writing, "
    "no calligraphy, no symbols, no marks, no stamps, no seals, no signature, "
    "no borders, no frame, no decorative patterns."
)
```

Generate one or two candidates, score them with `detect_text_in_image`, and stop early if the score is below `30.0`.

## Illustration Compositing

Use these three operations together:

- irregular corner radii: each corner uses a random radius around `0.30-0.45` of the shorter side
- feathered signed-distance alpha mask: process each corner only in its own bounding box
- paper tone blend: `image * 0.82 + paper_color * 0.18`

Composite with:

```python
canvas.paste(illustration_rgb, (illus_x, illus_y), alpha_mask)
```

The alpha mask should reveal the real background texture through transparent parts.

## Idempotent Batch Loop

```python
for item in records:
    output_path = output_dir / f"{item['idiom']}.png"
    if output_path.exists():
        continue

    cache_path = illustration_dir / f"{item['idiom']}_illustration.png"
    if not cache_path.exists():
        cache_path = generate_illustration(item, illustration_dir)

    try:
        generate_card(item, output_path, illustration_path=cache_path)
    except Exception as exc:
        failures.append((item.get("idiom"), str(exc)))
        continue
```

Use retrying, streaming image downloads for remote images. For large files, concurrent ranged downloads are useful, but a simple streaming retry loop is acceptable for small projects.

## Pitfalls

| Problem | Symptom | Fix |
| --- | --- | --- |
| AI illustration contains text | Chinese/calligraphy appears in picture | Keep prompt English, exclude title, score and retry |
| Illustration looks pasted on | hard rectangle or strong color mismatch | irregular corners, feathering, paper-tone blend |
| Alpha mask inverted | center transparent, corners opaque | process each corner bbox independently |
| Long story overflows | text runs off card bottom | adaptive font-size fitting |
| Pinyin misaligned | syllables drift left/right | center each syllable over its cell |
| Chinese character appears low | visual center below grid center | move text up by 10% of measured height |
| Network interruption | image download fails | streaming retry, optional ranged chunks |
| Missing glyphs | square boxes | fall back to a known CJK system font |

## Production Checklist

- Confirm background size and final export size.
- Confirm source records contain all fields and pinyin has been manually reviewed.
- Render short, medium, and long records.
- Verify repeated runs skip existing outputs and cached illustrations.
- Create final PNG folder, illustration cache folder, source data, and a listing/sales copy file when the product is for sale.
